# Zomato Data Analysis Dashboard

## Project Overview
This project analyzes Zomato restaurant data to uncover insights on customer preferences, pricing trends, and cuisine popularity. An interactive Power BI dashboard was built to support data-driven decision-making.

---

## ools & Technologies
- Power BI
- Excel
- DAX
- Power Query

---

## Key Features
- Interactive dashboard with filters and slicers
- KPI tracking (Average Rating, Total Restaurants, Cost for Two)
- City-wise and cuisine-wise analysis
- Rating vs Cost comparison

---

## Key Insights
- High-rated restaurants are concentrated in major cities
- Certain cuisines dominate customer preferences
- Higher cost does not always guarantee better ratings
